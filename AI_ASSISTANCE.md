# AI-assisted development

Parts of this project (parts of the documentation, some design ideas, and visual assets such as the logo) were created with assistance from a large language model.

The project logo particularly was generated with assistance from ChatGPT image generation tools.

The AI was used as a manual tool during development (similar to documentation search or brainstorming). It is **not** part of the runtime system, **does not** execute code, and **no** autonomous agents are used in production.

All final decisions, integration, and code review were done by the project maintainer.

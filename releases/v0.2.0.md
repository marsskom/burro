# Burro v0.2.0 — Release

## What's new

**BREAKING CHANGES!!!**

Burro, by default, uses zero configuration mode and is distributed without workdir example.

Plugin registration has been changed from auto-generated to manual and you may install Burro with `go install` command:

```sh
go install gitlab.com/marsskom/burro/cmd/burro@v0.2.0
```

Flag `-d %workdir%` now is required if you want to use Burro with configuration under %workdir%. Witout this flag it is always zero configuration mode.

### Initialization

New `burro init` command generates default working directory structure with some default configs inside.

It may be used to create basic workdir.
